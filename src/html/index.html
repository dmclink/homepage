About me

Currently pursuing a career in software development as can be seen by a few of the projects below (more on GitHub). I came from aerospace manufacturing which is a wildly different industry, but in both, I get take transform something from a thought to a functioning, performing machine in the real world. I hope to build safe and reliable software and applications that bring the same value to people as my little creations have to me.

I have experience using Go programming language to write backend servers. Python to collect and analyze data. PostgreSQL and SQLite relational databases. HTML, CSS, and JavaScript for frontend SPA's and interactive web pages. Linux on AWS and Heroku and my own devices. I've experimented with a number of different LLMs (self-hosted and cloud-based) for code generation, but most, if not all, of the code I've contributed to each of these projects was written by hand.

I'm currently learning React and Node.JS to round off the typical "full stack" skills necessary for fast development. I by no means feel limited by these skills and always looking to learn more to add to my toolbelt!

I contributed some "pages" to the frontend which is written in JavaScript and refactored the style and CSS for consistency.
